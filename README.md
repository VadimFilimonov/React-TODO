A simple todo list app built in React.

## Demo

http://vf-react-todo.surge.sh

## Setup

```sh
make install
```

## Build

```sh
make build
```

## Run

```sh
make develop
```

## Run lint

```sh
make lint
```

## Run tests

```sh
make test
```
